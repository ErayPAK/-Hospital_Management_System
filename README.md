Hospital Management System | Hastane Otomasyon Sistemi
Bu proje, hastane operasyonlarını dijitalleştirmek ve kullanıcı rollerine göre özelleştirilmiş bir deneyim sunmak amacıyla geliştirilmiş kapsamlı bir masaüstü otomasyon sistemidir.

Proje Hakkında (About the Project)
Proje, dört farklı kullanıcı tipinin (Admin, Doktor, Sekreter ve Hasta) ihtiyaçlarına yönelik modüller içerir. Veri yönetimi MS SQL Server ile sağlanırken, arayüz C# dili kullanılarak geliştirilmiştir.

Kullanıcı Rolleri ve Özellikler (User Roles & Features)
Admin: Sistem genelindeki kullanıcı yönetimi, doktor atamaları ve istatistiklerin takibi.

Doktor (Doctor): Hasta randevu listesi takibi, reçete oluşturma ve hasta geçmişi görüntüleme.

Sekreter (Secretary): Randevu oluşturma, hasta kayıt işlemleri ve duyuru yönetimi.

Hasta (Patient): Randevu alma, kişisel bilgileri güncelleme ve geçmiş randevuları inceleme.

Kullanılan Teknolojiler (Tech Stack)

Language: C# (.NET Framework/Core)
IDE: Visual Studio 2022
Database: MS SQL Server & SSMS (SQL Server Management Studio)
